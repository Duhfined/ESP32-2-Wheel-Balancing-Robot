# ESP32 2-Wheel-Balancing-Robot
A simple 2 wheeled robot capable of balancing on it's own.


Project Start Date: 2026-04-19


This is my first real project, and an introduction to robotics. I'm creating this project to not only build up my resume, but to have fun in the process and learn something outside of our scheduled education. I aim to explore the science behind IMU's and how a robot works in this proejct.
